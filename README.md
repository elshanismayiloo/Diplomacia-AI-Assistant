# Diplomacia AI Assistant

> AI-powered knowledge assistant for the Diplomacia online strategy game.

---

## Vision

Diplomacia AI Assistant (DAA) is an open-source artificial intelligence project designed to become the most comprehensive knowledge assistant for the Diplomacia community.

Instead of relying on predefined FAQ lists, DAA uses a structured Knowledge Base and a Diplomacia Knowledge Graph to understand concepts, relationships and game mechanics.

The assistant is designed to help both new and experienced players by providing fast, accurate and practical answers.

---

# Core Objectives

- Teach new players how Diplomacia works.
- Answer gameplay questions instantly.
- Explain game mechanics using official information.
- Provide practical recommendations without making strategic decisions for the player.
- Build the first structured Diplomacia Knowledge Graph.
- Support Telegram, Discord and future web applications.

---

# Project Philosophy

DAA never invents game mechanics.

Every answer must originate from the project's Knowledge Base.

Artificial Intelligence is responsible only for:

- Understanding the user's question
- Retrieving relevant knowledge
- Combining related concepts
- Generating a natural language response

The Knowledge Base remains the single source of truth.

# Diplomacia Knowledge Graph

Every concept is connected with related concepts.

Example:

Bilim Adamı

↓

Factory Income

↓

Economy

↓

Market

↓

Resources

↓

Buildings

↓

↓

Academic Career

The AI uses these relationships to produce better answers.
