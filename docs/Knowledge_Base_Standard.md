# Diplomacia AI Assistant
## Knowledge Base Standard (KB Standard v1.0)

---

# Purpose

This document defines the rules for creating every Knowledge Object inside the Diplomacia AI Assistant project.

Every concept, mechanic, guide and FAQ must follow this standard.

The objective is to keep the entire Knowledge Base consistent, searchable and AI-friendly.

---

# Core Principles

## 1. Single Source of Truth

Each concept exists only once.

Duplicate information is not allowed.

---

## 2. Atomic Knowledge

Each Knowledge Object explains only one concept.

Example:

✔ Bilim Adamı

✔ Akademik Kariyer

✔ Market

✘ Skills (contains multiple concepts)

---

## 3. Human Friendly

Every explanation must be understandable by a new player.

Avoid unnecessary technical language.

---

## 4. AI Friendly

Knowledge Objects must be structured.

Never write long essays.

Every section must have a clear purpose.

---

# Knowledge Object Structure

Every object contains the following sections.

## ID

Unique identifier.

Example

KB-0001

---

## Title

Official name of the concept.

Example

Bilim Adamı

---

## Category

High-level category.

Examples

Skills

Economy

Politics

War

Market

Factories

Regions

---

## Summary

One sentence definition.

Maximum:

40 words.

---

## Explanation

Detailed explanation.

Recommended length:

100–250 words.

---

## Practical Advice

How a player should use this knowledge.

---

## Common Mistakes

Typical beginner mistakes.

---

## Related Concepts

Links to other Knowledge Objects.

---

## Keywords

Search keywords.

Include:

Official names

Synonyms

Common spelling mistakes

Abbreviations

---

## Confidence

Allowed values

Official

Community

Experience

---

# Writing Rules

Always explain "what".

Then explain "why".

Finally explain "how".

---

# Forbidden

Do not repeat another Knowledge Object.

Do not include personal opinions.

Do not recommend political choices.

Do not recommend specific countries or alliances.
